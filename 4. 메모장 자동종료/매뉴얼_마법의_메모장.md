# 🌙 마법의 메모장 (Magic Notepad) 사용자 매뉴얼

## 1. 프로그램 개요
**마법의 메모장(Magic Notepad)**은 겉보기에는 세련된 다크 테마를 가진 평범한 메모장처럼 보이지만, 사용자가 특정 단어를 입력할 때만 작동하는 **숨겨진 비밀 기능(이스터에그)**을 포함하고 있는 특별한 프로그램입니다.

- **버전**: Premium Edition (1.0)
- **실행 파일 위치**: `dist/magic_notepad.exe`

---

## 2. 주요 디자인 특징
- **고급스러운 다크 네이비 테마**: 눈이 편안하면서도 개발자 친화적인 세련된 UI를 제공합니다.
- **테두리 없는(Borderless) 에디터**: 텍스트 입력의 몰입감을 극대화하기 위해 테두리를 없애고 넓은 여백(Padding)을 배치했습니다.
- **포인트 컬러**: 타이핑되는 커서와 상단 제목에 스카이 블루(Sky Blue) 컬러를 적용하여 직관적이고 아름다운 시각적 환경을 구축했습니다.

---

## 3. 핵심 기능 (이스터에그) 안내
이 프로그램의 진정한 매력은 마우스 조작이나 메뉴 버튼 클릭 없이, **오직 키보드 입력(타이핑)**만으로 특정 명령을 수행한다는 점입니다. 메모장에 자유롭게 글을 쓰다가 아래의 텍스트를 입력해 보세요.

### 🤫 비밀 기능 1. 즉각 숨김(자동 종료)
- **트리거 단어**: `종료` (띄어쓰기 없이 연속으로 입력)
- **작동 방식**: 다른 메모를 작성하다가 누군가 다가오거나 화면을 숨기고 싶을 때, 마치 자연스럽게 글을 쓰고 있던 것처럼 **'종료'**라는 두 글자를 입력하세요. 입력하는 즉시 저장 팝업 등 그 어떤 알림도 없이 프로그램이 메인 화면에서 흔적도 없이 완전히, 즉각적으로 종료됩니다.

### 🏫 비밀 기능 2. 비밀의 문 열기 (인하공전 공식 홈페이지)
- **트리거 단어**: 한글 `인하` 또는 영문 `inha`, `INHA`, `Inha` 등 (대소문자 무관)
- **작동 방식**: 해당 단어를 입력하는 순간 작성 중이던 해당 단어가 마법처럼 스르륵 지워집니다. 그와 동시에 기본 웹 브라우저가 자동으로 실행되며 **'인하공업전문대학' 공식 홈페이지(https://www.inhatc.ac.kr)**로 접속합니다. (페이지 무한 로딩을 막기 위해 입력된 트리거 단어는 시스템이 알아서 센스 있게 자체 삭제합니다.)

## 4. 실행 방법 및 주의사항
1. **실행 방법**: `dist` 폴더 안에 제공된 `magic_notepad.exe` 파일을 더블 클릭하여 실행합니다.
2. **설치 불필요**: 이 프로그램은 단일 포터블(Portable) 형태로 컴파일되었습니다. 파일 하나만 복사해서 USB에 담거나 이메일로 보내어 다른 PC에서도 바로 실행 가능합니다.
3. **절대 안전한 방식**: 외부 메모장(Notepad)의 메모리를 탈취하거나 키로거를 설치하는 방식이 아닙니다. 이 앱 자체적으로 내장된 에디터에서만 작동하므로 바이러스나 백도어 등의 보안 문제로부터 100% 안전합니다.
4. **저장 기능 없음**: 이 애플리케이션은 이스터에그 기믹을 즐기거나 타인으로부터 잠시 메모를 숨기기 위한 용도로 제작되었습니다. 별도의 텍스트 '저장' 버튼이 존재하지 않으므로 중요한 내용은 작성 후 메모장이 닫히기 전에 복사(Ctrl+C)하여 안전한 곳에 별도 보관하시길 권장합니다.
